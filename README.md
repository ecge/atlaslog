# AtlasLog

AtlasLog is a private travel journal and itinerary planner for tracking trips,
stops, budgets, photos, weather notes, and offline drafts.

## What it demonstrates

- Trip library with search and status filters.
- Route detail with a map-style stop preview.
- Budget progress, photo counts, weather context, and draft tracking.
- Travel app branding that stays original and inspectable.
- Responsive layout for planning on desktop or reviewing on mobile.

## Stack

- React 19
- TypeScript
- Vite
- Lucide React
- Plain CSS with reusable dashboard primitives

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
